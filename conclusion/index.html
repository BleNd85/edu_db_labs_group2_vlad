# Висновки

Під час проєктування та розробки сервісу для аналізу медіаконтенту було проведено глибокий аналіз предметної області. Визначено основні методи та підходи до вирішення цього завдання, а також створено детальну порівняльну характеристику нашої системи з існуючими рішеннями.

Якісне та ретельне проєктування інформаційного забезпечення дозволило реалізувати його у вигляді програмного забезпечення (Restful сервісу). Для імплементації цього продукту були використані технології MySQL та Java Spring Boot, що забезпечує масштабованість, підтримку та гнучкість системи до змін. Також було протестовано та впроваджено сучасну документацію до реалізованого API.
